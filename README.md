# Task Management

A personal task management application for organizing, tracking, and managing your daily tasks efficiently.

## 🎯 Overview

This is a lightweight, user-friendly task management web application built with HTML and JavaScript. It helps you stay organized and keep track of your tasks in one place.

## ✨ Features

- ✅ Create and manage tasks
- 📝 Simple and intuitive interface
- 💾 Easy-to-use task tracking
- 🎨 Clean, responsive design
- ⚡ Fast and lightweight

## 🛠️ Tech Stack

- **Frontend**: HTML, JavaScript
- **Hosting**: [Vercel](https://task-management-beige-eight.vercel.app)

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/panoskokmotos/task-management.git
cd task-management
```

2. Open `index.html` in your browser or deploy to your preferred hosting platform
